# F-005 Analysis Memory

## Intent
Preserve the analyst's evolving understanding. Notes, tags, renames, statuses, TODOs, and evidence links MUST be durable and attached to objects.

## Scope
- Object notes.
- Tags and review status.
- Function renames.
- TODO/hypothesis records.
- Evidence links and backlinks.

## Acceptance Signals
- Annotations survive project reopen.
- Annotated objects can be filtered.
- Inspector displays notes/tags/status.
- Evidence links can be promoted into findings.

## Dependencies
F-001, F-004.
